# 设计文档

本目录存放项目的设计文档，用于记录重大功能的架构设计和实现方案。

## 目录结构

```
design/
├── README.md           # 本文件
├── modules/            # 模块架构文档
├── draft/              # 草稿中的设计文档
└── archive/            # 归档的历史文档
```

## 使用方式

1. 创建新的设计文档时，复制 `template.md` 到 `draft/` 目录
2. 模块架构文档放在 `modules/` 目录

## 命名规范

- 文件名：`feature-name.md`
- 使用小写字母和连字符
- 例如：`editor-module.md`
